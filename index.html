🤖 Web Scraper de Notícias (G1)

Problema: A coleta manual de dados (manchetes de notícias) de sites é um processo lento, repetitivo e ineficiente para monitoramento de informações.

Solução: Um script de automação (Web Scraper) desenvolvido em Python que acessa o portal G1, analisa o código HTML da página e extrai automaticamente todas as manchetes principais.

Resultado: Dados coletados e organizados em um arquivo `.txt` em segundos, demonstrando habilidade prática com Python e as bibliotecas `requests` e `BeautifulSoup`.

🔗 Gerenciador de Links Full-Stack

Problema: A necessidade de gerenciar e organizar links pessoais (favoritos) de forma persistente, que seja mais robusta e customizável do que os favoritos do navegador.

Solução: Desenvolvimento de uma aplicação web full-stack completa. O back-end, construído em Python com Flask e SQLAlchemy, gerencia um banco de dados SQLite. O front-end permite que o usuário Crie, Leia, Edite e Exclua (CRUD) links de forma intuitiva.

Resultado: Um aplicativo web 100% funcional que demonstra o ciclo completo de desenvolvimento, integrando a interface do usuário, a lógica do servidor e a persistência de dados.
