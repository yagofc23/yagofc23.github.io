Sobre Mim

Olá, meu nome é Yago Fernandes da Costa, tenho 22 anos e sou um programador full stack júnior e analista de dados, apaixonado por criar soluções digitais e extrair valor de informações.

Com formação em Programação em Python pelo Senac, construí uma base sólida desenvolvendo projetos robustos com HTML, CSS, Python e Django. No universo dos dados, aplico Python, bibliotecas de análise e SQL para transformar informações brutas em insights estratégicos e métricas claras de negócio.

Estou disponível para projetos freelancers e oportunidades de trabalho no mercado de tecnologia e dados. Minha motivação é resolver problemas reais de forma inteligente, seja escrevendo códigos eficientes para a web ou estruturando análises e dashboards para entregar o melhor resultado.
